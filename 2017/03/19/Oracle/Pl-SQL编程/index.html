Pl/SQL编程

发表于 2017-03-19 | 分类于 学习笔记

pl/sql

优点

  1. 模块化编程
  2. 减少网络传输
  3. 提高运行性能
  4. 减少java编程的复杂度

特点

  1. 过程，函数，触发器是pl/sql编写的,存放在ORCALE中
  2. pl/sql是数据库过程语言
  3. 过程、函数可以在java程序中调用
  4. 安全性高，隐藏了表名等

缺点

  1. 移植性不好，只是适用orcale

开发工具

  1. sql_plus
  2. pl/sql developer
