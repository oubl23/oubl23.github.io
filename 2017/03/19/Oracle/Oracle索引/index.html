Oracle索引

发表于 2017-03-19 | 分类于 学习笔记

原则

  1. 为经常查询的字段创建索引，不经常查询的字段不用索引
  2. 大表建立索引才有意义
  3. 经常结合where的字段加索引
  4. 不超过4个

缺点

  1. 占用更大的空间(约1.2倍)
  2. 更新数据的时候会使用更多的时间
